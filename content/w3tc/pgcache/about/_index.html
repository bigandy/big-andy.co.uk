About me

My name is Andrew Hudson and I live in Reading, UK with my wife Marion and two kittens Inti and Punku.

I work for HeathWallace in Reading as a creative front-end developer.

Specialising in WordPress theme and plugin development I love working with HTML5, CSS3, PHP and JavaScript (currently learning this).

In my spare time I a regularly go cycling, take photos, listen to music and go to the cinema.

Colophon

This site is powered by WordPress using WP Skeleton as a template. The theme I’m using is built on top of the Bones starter theme. CSS is pre-compiled by Sass.

I’m tracking the code on Github and deploying with Mark Jayquith’s WP Stack, a customised version of Capistrano. The site is hosted in the AWS cloud in Ireland.

For the typography I am using Chaparral Pro for the headings and Freight Sans Pro for the body, both served by Typekit.
