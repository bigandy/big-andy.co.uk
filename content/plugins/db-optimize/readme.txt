== Description ==

You can use DB - Optimize to reclaim the unused space and to defragment the database.
It does not require PhpMyAdmin.

For your tables, DB - Optimize works as follows: 

* Repairs the table, if it has deleted rows
* Sorts the index pages
* Updates the table's statistics, if not up to date

Support languages: english, german and french.

== Installation ==

1. Uncompress the download package
2. Upload folder including all files and sub directories to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress

You're done!

Start optimize your database using Menu 'Tools > DB-Optimize'.

== Frequently Asked Questions ==

= Will I have to configure the plugin ? =

No, automatic detection of the database tables and optimization.
